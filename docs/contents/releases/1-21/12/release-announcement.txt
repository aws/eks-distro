Amazon EKS Distro v1.21-12 is now available. This release includes an update to Amazon Linux 2 that contains the latest security fixes, a new patch to Kubernetes, and a new version of aws-iam-authenticator. Amazon EKS Distro v1.21-12 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
