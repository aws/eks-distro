Amazon EKS Distro v1.21-7 is now available. This release includes an upgrade from Kubernetes v1.21.2 to v1.21.5, a number of changes to the Kubernetes patches, newer versions of several components, and an update to Amazon Linux 2, which contains the latest security fixes. Amazon EKS Distro v1.21-7 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
