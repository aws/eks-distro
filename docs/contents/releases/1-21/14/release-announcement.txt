Amazon EKS Distro v1.21-14 is now available. This release includes a new Kubernetes patch version, updates to the patches, an update to Amazon Linux 2 that contains the latest security fixes and updates to some project versions and project dependencies. Amazon EKS Distro v1.21-14 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
