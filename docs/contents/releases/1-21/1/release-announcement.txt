Amazon EKS Distro v1.21-1 is now available. This release now supports Kubernetes v1.21.2. Additional highlights of this release include the latest version of Amazon Linux 2 and upgraded versions of some Kubernetes components, such as etcd, metrics-server, and pause. Amazon EKS Distro builds of Kubernetes 1.21-1 are available through ECR Public Gallery (https://gallery.ecr.aws/?searchTerm=EKS+Distro) and GitHub (https://github.com/aws/eks-distro)
