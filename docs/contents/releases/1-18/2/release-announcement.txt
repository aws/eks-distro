Amazon EKS Distro now supports Kubernetes version 1.18.2. Highlights of the Kubernetes 1.18.2 release include the deletion of leaked volumes and proper de-registration of cordoned nodes from target groups. Amazon EKS Distro builds of Kubernetes 1.18.2 are available through ECR Public Gallery (https://gallery.ecr.aws/?searchTerm=EKS+Distro) and GitHub (https://github.com/aws/eks-distro)
