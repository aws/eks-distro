Amazon EKS Distro v1.18-13 is now available. This release includes a number of changes to the Kubernetes patches, a newer version of aws-iam-authenticator, and an update to Amazon Linux 2, which contains the latest security fixes. Amazon EKS Distro v1.18-13 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
