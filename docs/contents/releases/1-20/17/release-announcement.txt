Amazon EKS Distro v1.20-17 is now available. This release includes a new patch to Kubernetes, an update to Amazon Linux 2 that contains the latest security fixes, and updates to some project versions and project dependencies. Amazon EKS Distro v1.20-17 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
