Amazon EKS Distro v1.20-12 is now available. This release includes an update to Amazon Linux 2 that contains the latest security fixes, a new patch version of Kubernetes (v1.20.15), and updates to many project versions. Amazon EKS Distro v1.20-12 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
