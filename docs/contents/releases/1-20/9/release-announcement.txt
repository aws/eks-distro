Amazon EKS Distro v1.20-9 is now available. This release includes an upgrade from Kubernetes v1.19.13 to v1.19.15, a number of changes to the Kubernetes patches, a newer versions of aws-iam-authenticator and Metrics Server, and an update to Amazon Linux 2, which contains the latest security fixes. Amazon EKS Distro v1.20-9 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
