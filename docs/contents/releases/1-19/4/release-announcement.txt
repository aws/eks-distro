Amazon EKS Distro v1.19-4 is now available. This release includes upgraded versions of Kubernetes (from v1.19.6 to v1.19.8) and of Amazon Linux 2, which contains the latest security fixes. Amazon EKS Distro v1.19-4 builds of Kubernetes are available through ECR Public Gallery (https://gallery.ecr.aws/?searchTerm=EKS+Distro) and GitHub (https://github.com/aws/eks-distro)
