Amazon EKS Distro v1.19-15 is now available. This release includes an update to Amazon Linux 2 that contains the latest security fixes, a new patch version of Kubernetes (v1.19.16), and updates to many project versions. Amazon EKS Distro v1.19-15 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
