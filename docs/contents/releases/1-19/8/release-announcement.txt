Amazon EKS Distro v1.19-8 is now available. This release includes upgraded versions of Kubernetes (from v1.19.12 to v1.19.13) and an update to Amazon Linux 2, which contains the latest security fixes. Amazon EKS Distro v1.19-8 builds are available through ECR Public Gallery (https://gallery.ecr.aws/eks-distro) and GitHub (https://github.com/aws/eks-distro)
