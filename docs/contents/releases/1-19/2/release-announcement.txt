Amazon EKS Distro now supports Kubernetes version 1.19-2. Highlights of the Kubernetes 1.19-2 release include the deletion of leaked volumes, additional endpoint options for ECR, and a newer release of Amazon Linux 2. Amazon EKS Distro builds of Kubernetes 1.19-2 are available through ECR Public Gallery (https://gallery.ecr.aws/?searchTerm=EKS+Distro) and GitHub (https://github.com/aws/eks-distro)
